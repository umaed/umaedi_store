## 🎨 CUSTOMIZATION QUICK START

### 1. Add Product Images
```
1. Find: assets/img/ folder
2. Add your images (JPG or PNG)
3. Size: ~200x200px (optimal)
4. Edit: js/product.js
5. Replace: placeholder paths with your images
6. Done!
```

### 2. Update Product List
```
File: js/product.js
Find: const products = [...]

Add new product:
{
  id: 'unique-id',
  name: 'Your Product',
  price: 500000,
  img: '../assets/img/your-image.jpg',
  category: 'fashion',
  rating: 4.8
}
```

### 3. Change Store Info
```
File: js/config.js or js/footer.js

- Store name
- Owner info
- WhatsApp number
```

## 🐛 TROUBLESHOOTING

### Problem: "Cannot find index.html"
**Solution**: Make sure you're in correct folder
```
cd "c:\Users\umaed\OneDrive\Dokumen\umaedi store"
node server.js
```

### Problem: Images not loading
**Solution**: Check image paths in product.js
- Make sure images in assets/img/
- Check filename spelling
- Ensure correct path: ../assets/img/image.jpg

### Problem: Cart not saving
**Solution**: 
- Clear browser cache
- Try different browser
- Check localStorage enabled
- Incognito mode doesn't persist

### Problem: WhatsApp link not working
**Solution**:
- WhatsApp must be installed
- Check internet connection
- Phone number format: 6283818115136
- Try on mobile device

### Problem: Console errors
**Solution**:
- Check browser console (F12)
- Read error messages carefully
- Search error online
- Check file paths
- Verify all files present
